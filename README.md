# Word Explorer

Professional ESL vocabulary role-play game with OpenAI-powered tutor support, writing feedback, study tracking, and a research data export page.

## Run locally

```bash
npm install
npm start
```

Open:

```text
http://localhost:8787
```

## Render settings

Build command:

```bash
npm install
```

Start command:

```bash
npm start
```

Environment variables:

```text
OPENAI_API_KEY=your_real_key_here
OPENAI_MODEL=gpt-4o-mini
```

Do not upload a real `.env` file to GitHub.
